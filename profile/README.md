# CKM Tools

Small, sharp tools for developers who ship.

---

## Products

### [textlens](https://ckmtools.dev/textlens/) — Text analysis for Node.js

Zero-dependency TypeScript library. Readability scoring (8 formulas), sentiment analysis, keyword extraction, SEO scoring — one function call.

[![npm version](https://img.shields.io/npm/v/textlens?color=00E5A0&style=flat)](https://npmjs.com/package/textlens)
[![npm downloads](https://img.shields.io/npm/dw/textlens?color=00E5A0&style=flat)](https://npmjs.com/package/textlens)
[![license](https://img.shields.io/npm/l/textlens?style=flat)](https://github.com/ckmtools/textlens/blob/main/LICENSE)

`npm install textlens`

---

### [prosescore](https://prosescore.ckmtools.dev/) — Browser-based readability analyzer

Privacy-first readability checker. Runs entirely client-side — nothing sent to any server. 8 readability formulas with consensus grading.

[![GitHub stars](https://img.shields.io/github/stars/ckmtools/prosescore?color=00E5A0&style=flat)](https://github.com/ckmtools/prosescore)
[![license](https://img.shields.io/npm/l/prosescore?style=flat)](https://github.com/ckmtools/prosescore/blob/main/LICENSE)

[Try it at prosescore.ckmtools.dev](https://prosescore.ckmtools.dev/)

---

### [TextLens API](https://ckmtools.dev/api/) — Hosted text analysis REST API

Same analysis as textlens, hosted. Works in Python, Ruby, Go, PHP — any language with HTTP. Free tier: 1,000 requests/month.

[Get started at ckmtools.dev/api/](https://ckmtools.dev/api/)
