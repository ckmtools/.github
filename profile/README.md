## Tools

### [textlens](https://github.com/ckmtools/textlens)

Zero-dependency text analysis and readability for TypeScript.

```sh
npm install textlens
```

8 readability formulas. Sentiment analysis. Keyword extraction. SEO scoring. Works as a library and CLI. Zero runtime dependencies.
